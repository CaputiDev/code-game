# CodeGame 🎮

**Jogo educativo 2D de plataforma/puzzle para ensino de lógica de programação.**

> Desenvolvido como Trabalho de Conclusão de Curso (TCC) - IFSUL Charqueadas.
> Engine: **Godot Engine 4.3** | Linguagem: **GDScript** |
